Drop GLB files here so we can wire them into the game quickly.

Recommended:
- keep each model and its textures in the same folder tree under `public/models`
- use simple names such as `train.glb`, `barrier-top.glb`, `barrier-low.glb`
- if a model depends on bin/textures, keep the relative paths unchanged

Examples:
- `public/models/train.glb`
- `public/models/barrier-top.glb`
- `public/models/barrier-low.glb`
